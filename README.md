# Banking Management System

## Project Description
This is a basic Banking Management System project developed using Core Java.
It is a console based and menu driven application.
I created this project to practice Java basics and OOP concepts.
Even though this project is simple, I tried my best to implement it properly.

## Author
Vaishnavi Bandaru

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- Menu driven program
- Exception handling

## Concepts Used
- Core Java
- OOP concepts
- Classes and Objects
- Exception handling

## Tools Used
- Java
- VS Code
- GitHub

## How to Run the Project
1. Open the project in any Java IDE
2. Run the main file (BankApp.java)
3. Follow the menu shown in the console
